## Made with ❤️ in React and Redux
### Features:
  - See Current Weather of your City
  - See 5 days forecast of your selected City
  - A static Login form in order to checkOut the weather.

Install : 
```
npm install
```

Launch : 
```
npm start
```

### Thanks to OpenWeather API
